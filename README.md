## 作成した目的
模擬案件の開発演習をするため（初級編）

## 機能一覧
会員登録、ログイン、ログアウト、勤務開始、勤務終了、休憩開始、休憩終了、日付別勤怠情報取得、ページネーション

## 使用技術（実行環境）
Laravel 8.x
